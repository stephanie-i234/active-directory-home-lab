# Active Directory Domain Services Deployment

## Objective

Deploy Active Directory Domain Services (AD DS) and establish the foundational domain structure for the lab environment.

---

## Configuration

* Installed the Active Directory Domain Services (AD DS) role.
   ![ADDS Installation and Config](../screenshots/ADDS%20Installed.PNG)
* Promoted the server to a Domain Controller.
  
* Created the **corp.local** domain.
  
* Configured the following Organizational Units (OUs):
  * IT
  * Human Resources (HR)
  * Sales
  
* Created domain user accounts.
  ![oraganizational units shown](../screenshots/organizational%20units.PNG)
* Created security groups to support user and resource management.

---

## Outcome

A functional Active Directory environment was successfully deployed. The **corp.local** domain was operational, with Organizational Units, user accounts, and security groups configured to support administrative and help desk tasks throughout the lab.
