# Deploying the Domain Controller

## Objective

Deploy a Windows Server 2022 virtual machine to serve as the Domain Controller for the Active Directory lab environment.

## Configuration

* Created a new virtual machine in VMware Workstation.
* Installed Windows Server 2022.
* Configured virtual machine resources (CPU, memory, storage, and network adapter).
* Renamed the server to **DC01**.
   ![server renamed image](screenshots/server_name_change.png)
* Verified the operating system installation completed successfully.
  ![Confirmed Server Name Change](screenshots/local_server_screenshot.PNG)

## Outcome

The virtual machine was successfully deployed and prepared for Active Directory Domain Services installation.
